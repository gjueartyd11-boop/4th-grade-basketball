## 세트차 수정
순위표의 게임차를 세트차로 변경했습니다. 세트차 = 세트승 - 세트패

## 이번 수정
- 순위표의 승/무/패를 경기 기준이 아니라 세트 기준 누적으로 표시합니다.
- 세트차는 1위 기준으로 필요한 세트 수를 표시합니다.
- 세트차 공식: ((1위 세트승 - 해당 반 세트승) + (해당 반 세트패 - 1위 세트패)) ÷ 2
- 마지막 세트기록 컬럼은 제거했습니다.

## 모바일 표 수정
- 연속 칼럼 삭제
- 순위표가 옆으로 스크롤되지 않도록 compact 레이아웃 적용

## 세트차 계산 변경
세트차를 1위 기준 세트점수 차로 변경했습니다.

세트점수 = 세트승 + 세트무 × 0.5
세트차 = 1위 세트점수 - 해당 반 세트점수

## 1위까지 컬럼 수정
- `세트차` 대신 `1위까지`로 표시합니다.
- 세트점수 = 세트승 + 세트무 × 0.5
- 1위까지 = ceil(1위 세트점수 - 해당 반 세트점수)
- 1위는 `-`로 표시합니다.

## 공동 순위 및 1위까지 계산 수정
- 승률이 같은 반은 같은 순위로 표시합니다. 예: 1위, 1위, 3위
- `1위까지`는 1위 승률에 도달하기 위해 필요한 연속 승리 세트 수입니다.
- 계산식: 현재 세트점수 S, 전체세트 T, 1위 승률 R일 때
  필요 세트 수 = ceil((R×T - S) / (1 - R))
- 1위 승률이 1.000이고 따라잡을 수 없는 경우 `불가`로 표시합니다.

## 승점 컬럼 수정
- `1위까지` 컬럼을 삭제하고 `승점` 컬럼으로 변경했습니다.
- 승점 = 세트승 + 세트무 × 0.5
- 공동 순위 표시는 유지합니다.


## 승점 계산 수정
승률은 그대로 유지합니다.
승률 = (세트승 + 세트무×0.5) ÷ 전체세트

승점은 아래 방식으로 변경했습니다.
승점 = 세트승 + 세트무×0.5 - 세트패

## 필요세트차 추가 (4학년 넷볼)
- 순위표에 `필요세트차`를 추가했습니다.
- 의미: 남은 경기에서 1위 평균승점에 도달하려면 필요한 총 세트 우위입니다.
- 계산식 안내는 순위표 하단에 3줄로 정리했습니다.

## 이번 수정
- 필요세트차 컬럼을 삭제했습니다.
- 순위 정렬 기준을 `승률 → 평균승점 → 세트득실 → 세트승` 순서로 바꿨습니다.
- 공동순위는 승률과 평균승점이 모두 같을 때만 같은 순위로 표시됩니다.
